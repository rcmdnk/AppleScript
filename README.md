AppleScript
===========

# Installation

Below commands will make scpt files in $HOME/Library/Scripts/ (user's script folder)
Please be careful if you have same name files...

    cd ~/tmp
    git clone git@github.com:rcmdnk/AppleScript
    cd AppleScript
    ./install.sh
